# YouTube Toxic Comment Filter

## 🔥 Project Description
A Chrome extension that detects and hides toxic comments on YouTube using Machine Learning.

## 🚀 Features
- Real-time comment filtering
- ML-based toxicity detection
- Chrome extension integration

## 🧠 Technologies Used
- Python (Flask)
- Machine Learning (Logistic Regression + TF-IDF)
- JavaScript (Chrome Extension)

## ⚙️ How to Run

### Backend
cd backend
python app.py

### Extension
1. Open Chrome
2. Go to chrome://extensions
3. Load unpacked → select extension folder

## 📊 Dataset
Jigsaw Toxic Comment Dataset (Kaggle)
